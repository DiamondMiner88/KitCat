## Running
Only for arm based systems (ie. a RasPi) do the following two commands
```
$ sudo apt-get update 
$ sudo apt-get install build-essential libcairo2-dev libpango1.0-dev libjpeg-dev libgif-dev librsvg2-dev
```

Install dependencies
```
npm i
```

Run bot with API
```
npm run start
```
